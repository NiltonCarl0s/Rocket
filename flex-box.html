## Flexbox A caixa se torna flex, fazendo com que os elementos internos possam receber melhor: - Alinhamento - Ordenação - Tamanhos flexiveis # Terminologia - Flex Container - Flex item - Nesting > são elementos dentro de outro elemento - Axis > é o eixo - main - start, end - cross - start, end - Flex sizing - flexível - altera width/height dos itens para preenchimento dos espaços do flex container
