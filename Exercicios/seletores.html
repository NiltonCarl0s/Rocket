# Seletores Conecta a um elemento HTML com CSS a fim de alterar o elemento. ## Tipos * Element selactor - Todos os elementos HTML * ID Selector - Um elemento que tenha um atributo `id`. * Class Selector - Os elementos que contenha um atributo `class`. - Podemos ter uma ou mais classes. * Atribute Selector - Um elelmento que tenha um atributo especifico. * Pseudo-class Selector - Elementos em um estado especifico. ## Múltiplos Você poderá selecionar múltiplos elementos e aplicar alguma regra css para todos eles. Usamos uma separação por vírgula para isso. ```CSS h1, p, a { color: red: } ``` Document
