# Pseudo-classes É um tipo de seletor que irá selecionar um elemento que estiver em um estado especifico. Exemplo: É o primeiro elemento dentro de uma caixa, ou, o elemento está com o ponteiro do mouse em cima dele. Pseudo-classes começam com 2 pontos seguido do nome da pseudo class `:pseudo-class-name` ## Selecionando elementos com Pseudo-classes * :first-child * :nth-of-type() * :nth-child()
